## 🎯 Domain Usage in Glossa CAT

### For Translators:
- Select domain when annotating segments
- Helps identify specialized terminology
- Improves translation quality

### For Reviewers:
- Verify domain-specific accuracy
- Check terminology consistency
- Ensure style matches domain conventions

### For AI Training:
- Domain-tagged datasets improve AI models
- Enables domain-specific fine-tuning
- Better translation quality for specialized content

### For Project Management:
- Route jobs to specialized translators
- Set appropriate rates by domain
- Track translator expertise
